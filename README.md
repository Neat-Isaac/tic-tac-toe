# tic-tac-toe
- This is an inteligent tic-tac-toe.
- Organized, package system.
- Terminal based ui system.
- Randomization based algorithm for playing against the bot.

## Installation
Open git bash
```
$ git clone https://github.com/Neat-Isaak/tic-tac-toe.git
```
How to run
```
python test.py
```
